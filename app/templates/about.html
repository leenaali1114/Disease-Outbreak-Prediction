About the Project

Learn more about the Malappuram Disease Outbreak Prediction System.

Project Overview

The Malappuram Disease Outbreak Prediction System is a machine learning-based application designed to predict and prevent disease outbreaks in the Malappuram district of Kerala, India. By analyzing historical data on disease occurrences, environmental factors, and demographic information, the system can forecast potential outbreaks and provide preventive recommendations.

Key Features

  • Predictive Analytics: Advanced machine learning models to predict disease outbreaks with high accuracy.
  • Interactive Dashboard: Comprehensive visualization of disease patterns and statistics.
  • Geographical Mapping: Visual representation of disease hotspots and risk areas.
  • AI-Powered Recommendations: Preventive measures generated using Groq's advanced LLMs.
  • Email Alert System: Automated notifications for high-risk predictions.

Technology Stack

  • Backend: Flask (Python web framework)
  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • Data Visualization: Chart.js, Leaflet.js
  • AI Integration: Groq API for preventive recommendations
  • Email Service: Flask-Mail for alert notifications

Dataset

The system uses a comprehensive dataset of disease outbreaks in Malappuram district, including information on:

  • Disease types (Cholera, Dengue, Malaria, COVID-19, etc.)
  • Locations within Malappuram
  • Number of cases and deaths
  • Environmental factors (rainfall, temperature, humidity)
  • Infrastructure metrics (hospital beds, vaccination rates)
  • Historical outbreak patterns
